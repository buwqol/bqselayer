# bqselayer
Base layer for my (larger) C projects

## Features

- Context cracking
	- Language (C/C++)
	- Language standard year
	- Compiler (Clang/MSVC/GNU C)
	- Operating system (FreeBSD/Linux/Windows/MacOS/MSDOS/Unix)
	- Architecture (AMD64/Intel86/Arm64/Arm)
	- 'Bitness' (i.e. 64bit/32bit)
- Helper macros
- Typedefs and related functions
- Compound types (vectors, matrices)
